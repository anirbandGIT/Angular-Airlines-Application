About

This application is made using Angular v8. App also uses material design components from Angular Material v8 library. JSON Server is used to fake REST API.

This application allows airline staff and admin to securely login and manage airline services. Staff can check-in for passengers and manage in-flight itinerary services like choosing meals, extra luggage and adding in-flight shop items that will be billed to the customer. Admin can manage the flight ancillaries including adding/ removing shop items, min/max luggage weight restrictions, in flight menu items and also passenger data, including adding new passengers.

Running the App

Before running the application run the server using command npm run server in the terminal. To run the application run npm start in terminal.
